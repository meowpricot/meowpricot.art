## Accessibility
This website (meowpricot.art) was created, to the best of the creator's ability, to be accessible following the [Web Content Accessibility Guidelines](https://www.w3.org/TR/WCAG22/).

<br>

I have used the [WAVE Browser Extension](https://wave.webaim.org/) to check this website for accessibility errors. I have also gone through the [Easy Checks](https://www.w3.org/WAI/test-evaluate/preliminary/) method for further checking.

<br>

### Implemented accessibility features
This site has the following checks and features implemented:
- Layout still functions up to 200%
- Mobile-responsive layout
- Alt text for images when descriptions are needed, otherwise hidden
- Body font colour meets AA contrast levels
- Semantic HTML5 with clear headings

### Features that are currently missing
This site is missing these features, but are planned to be added in the future:
- Dark mode layout
- Skip to main content & jump to top of the page
